微信小程序 invalid code 错误

2019-06-28

做项目时跟前端对接登录接口 出现invalid code问题。关键是我可以正常获取，前端童鞋那里就报错，百思不得姐的同时让uniapp背了锅。后来猛然间想起，有可能是开发者工具使用的appid的问题，果不其然。
　　前提是后台接口使用的是我本人的测试账号，我开发者工具使用的自然也是我自己的测试账号。但是前端童鞋用的则是他自己的测试账号。这就导致了微信报错invalid code。因为code是微信服务器给appid分配的。前端appid的code自然不能作为后端appid的code去使用。
解决办法
修改一下你项目的appid即可
